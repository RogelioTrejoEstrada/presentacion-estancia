Problem Statement

The company MaunaKea, Digital Agency; does not have a portal as a business tool for better interaction with the customer, so it will be creating the same in order to improve interaction with the customer and meet the goal of the company

Objectives/Project Scope

Main objective

  • Beta Version Implementation (PreBeta, Beta and PostBeta)

Secondary objectives

  • Implementation of RESTfull Api for image uploads
  • Review new code implementations for proper integration to the system (Code Review)
  • Manage Server and security issues (Cloud)
  • Support the development team on technical issues of the system (Technical Lead)
  • Fulfill tasks assigned by the Project Leader
